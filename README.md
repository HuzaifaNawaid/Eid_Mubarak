# Eid Mubarak Pen ✨🌙

A web-based interactive drawing application that lets you draw glowing, festive gold ink in the air using your webcam and hand tracking. Built using HTML5 Canvas and Google's MediaPipe.

## ✨ Features

* **Real-time Hand Tracking**: Accurately tracks your hand movements using MediaPipe Hands.
* **Air Drawing**: Use your index finger as a virtual pen.
* **Glowing Ink Effect**: Draws with a smooth, glowing gold aesthetic perfect for Eid greetings.
* **Motion Smoothing**: Applies linear interpolation to your finger movements for perfectly smooth lines.
* **Mirror Display**: Horizontally mirrored canvas for intuitive, mirror-like interaction.

## 🚀 How to Run

1.  Download the `index.html` file
2.  Ensure you have a webcam connected to your computer.
3.  Open the `index.html` file directly in any modern web browser (Chrome, Firefox, Edge, Safari).
4.  Grant the browser permission to access your webcam.

## 🎮 Controls

* **Draw**: Press and hold the **`Shift`** key while moving your index finger around the camera view.
* **Clear**: Press the **`Spacebar`** to clear the canvas and start over.
